## Instrucciones

* Abra su Terminal y cree un nuevo folder, el nombre del folder debe ser **taller-git**.
* En el folder **taller-git**, cree un nuevo repositorio (git init).
* Usando como base la tarea de investigar un personaje, cree un archivo HTML que contenga: foto, nombre completo, un extracto de su biograf&iacute;a y link a redes sociales.
* Debe utilizar CSS para crear un layout de una sola columna.
* La estructura esperada es: un folder para el CSS, un folder para imagenes y un archivo HTML.
* Su repositorio debe tener dos braches. Un branch debe tener el nombre de **master** y debe estar vacio, el segundo branch se debe llamar **personaje** y debe contener los archivos y folders antes indicados.
* El branch debe tener un commit con los cambios relacionados, el mensaje debe ser "[Taller] Agrego estructura base para taller de Git"
* Cree un repositorio en GitHub con nombre **taller-git**, y haga push de sus cambios locales al repositorio.
* Haga un fork del repositorio [https://github.com/jleiva/git-practica.git](https://github.com/jleiva/git-practica.git), clone el nuevo repositorio y edite el archivo **lista-personajes.json** agregando una nueva entrada, indicando su nombre y la direcci&oacute;n de su repositorio **taller-git**
* Haga push de sus cambios y abra un pull request.
